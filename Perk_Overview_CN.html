四、终极组合 TOP3

🥇 第一名：超级坦克

口渴巨人 + 补水专家 + 身经百战老兵

  • 生命力 +100%
  • 零负效果
  • 最强肉盾，适合前线战斗单位

🥈 第二名：力量战士

熊的力量 + 铁胃 + 强力活力

  • 力量 +25% +10
  • 耐力 +25%
  • 近战经验 +0.25
  • 零负效果
  • 最强近战输出

🥉 第三名：终极采集

虾米伐木工 + 铁胃 + 伐木工的力量 或 石头专家 + 铁胃 + 碎石者

  • 专精采集双倍收益
  • 经验获取翻倍
  • 零负效果
  • 最强资源采集

五、注意事项

  1. 铁肺慎用：睡眠恢复-25%目前无法对冲，会导致村民频繁疲劳
  2. 季节福利：冬日猎手、夏季垂钓者等仅在特定季节生效，注意安排工作
  3. 环境福利：雨舞者、风暴赋能等依赖天气，收益不稳定
  4. 铁胃是核心：能对冲4个常用福利的负效果，优先招募
